
## Stack

- astro
- typescript
- react
- tailwind
- react-icons
- shadcn/ui
- prettier
- locomotive-scroll

## Install dependencies:

```bash
npm install
```

## Run the development server:

```bash
npm run dev
```

Open up [http://localhost:4321/](http://localhost:4321/)
